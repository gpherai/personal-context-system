# Personal Context System

Personal Context System is a planned single-user application for capturing, structuring, connecting, and using Gerald's personal context with AI.

This repository starts as an architecture/design workspace. Implementation should wait until the product model, data architecture, UI/UX direction, and AI access model are intentionally designed.
